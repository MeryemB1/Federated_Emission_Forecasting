# Carbon Footprint Estimation Project

This project aims to estimate carbon footprint in traffic environments using deep learning techniques. The system includes spatiotemporal forecasting to provide real-time emissions data.

## Features
- Spatiotemporal forecasting for emission trends
- CL Federated learning among road clients
- Edge deployment optimization

## Technologies Used
- Python, PyTorch, TensorFlow
- Flower
